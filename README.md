# Project Evo

Because I can.

## Install

	git clone git@github.com:alexeyraspopov/evo.git
	cd evo
	npm install
	make start

## Goals

 * Infrastructure
   * Tests
   * Coverage
   * Code style
   * Hot module replacement
   * Isolated modules environment
 * Client
   * Material design
   * CSS Modules (parcelify, css-modules, should be hot-reloaded)
   * As much stateless components as possible
   * http://ogp.me/
 ...
